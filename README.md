# 📈 CryptoPredict: AI-Based Time Series Analysis

An end-to-end Machine Learning pipeline designed to forecast cryptocurrency prices and analyze market sentiment. Developed as part of my Data Science internship.

## 🚀 Project Overview
This project addresses the high volatility of the crypto market by combining traditional statistical models with modern Deep Learning architectures. It provides users with an interactive dashboard to visualize trends, predicted prices, and real-time market "mood."

## 🛠 Features
* **Data Collection:** Automated ETL pipeline using `yfinance` for real-time market data.
* **Forecasting Models:** * **ARIMA:** For short-term statistical trends.
    * **Prophet:** To handle seasonality and market shocks.
    * **LSTM (RNN):** For capturing long-term dependencies in price sequences.
* **Sentiment Analysis:** NLP-based scraping of news headlines to gauge market fear/greed.
* **Interactive GUI:** Built with **Streamlit** for a seamless user experience.
